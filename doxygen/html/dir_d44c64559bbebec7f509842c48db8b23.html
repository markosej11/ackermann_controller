file  controller.hpp [code]
 This is the core of the software which makes use of other classes to calculate the headings and wheel speeds for the front wheels. We first find the error in heading and speeds and the use a PID controller to calculate the increment needed. Using this we use the Ackermann method to calculate the heading and speeds for the front wheels. Finally using the wheel headings and speed we calculate the speed and heading of the robot in global frame.
